# IMDB Movie Review Sentiment Analysis (LSTM)

This project uses a deep learning model to classify **IMDB movie reviews** as **Positive** or  **Negative** .

The model is built with an **Embedding + LSTM architecture** in TensorFlow/Keras and is trained on the IMDB dataset.

---

## Project Overview

* **Task:** Binary Sentiment Classification
* **Dataset:** IMDB Large Movie Reviews
* **Model:** LSTM-based neural network
* **Purpose:** Predict sentiment of movie reviews for analysis or application integration
